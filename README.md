# USVControl

2021.11.30

#### 介绍

python版船控，也可控两轮无人车，作者初学python练手。

#### 使用说明
​		目前仅有遥控功能，程序跑在工控机上，遥控器接收机S.BUS2接口连接COM50，同时COM50向外发送修改后的S.BUS2协议控制电机。中间信号转换可借助FT232模块、接收机信号转换模块，切记S.BUS协议需硬件取反。
